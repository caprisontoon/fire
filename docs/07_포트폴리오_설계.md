# [7차 산출물] 포트폴리오 설계 모드

> 상태: **구현 완료** — `backend/portfolio.py`, `public/portfolio.html`
> 배경: 기존 "현재 상태 진단"은 배당·이자 액수를 직접 입력받지만, 사용자는 **상품 구성을 바꿔가며
> 노후 현금흐름·세금·건보료를 설계·비교**하고 싶어 함. 이 모드는 입력 방향을 뒤집는다.

## 1) 무엇이 다른가

| | 현재 상태 진단 (index) | 포트폴리오 설계 (portfolio) |
|--|--|--|
| 입력 | 배당·이자 **금액**을 직접 | **상품 유형 + 평가금액 + 수익률** 라인 |
| 금융소득 | 사용자가 계산해 입력 | 시스템이 상품별로 자동 산출·분류 |
| 비교 | 금융소득 자동 축소(A/B/C) | 사용자가 만든 구성 2~3개 |
| 목적 | 지금 내 상태 진단 | "어떻게 담을지" 의사결정 |

## 2) 상품 유형 → 세금 분류 (핵심)

`backend/portfolio.py`의 `PRODUCT_TYPES`가 상품 유형마다 산출 소득이 어느 금융소득 칸으로
들어가는지 정한다. 사용자는 유형만 고르면 된다.

| 유형 | 연 소득 = 금액 × 수익률 | 금융소득 분류 |
|------|------------------------|---------------|
| 국내 주식(배당주) | 배당 | 배당소득 ✅ |
| 국내 주식형 ETF | 분배금 | 분배금만 ✅ (매매차익 비과세) |
| 기타 국내 ETF | 분배금+과세 매매차익 | 전액 ✅ |
| 해외 주식 | 배당 | 배당소득 ✅ (양도차익 별도) |
| 해외 상장 ETF | 분배금 | 분배금 ✅ (양도차익 별도) |
| 채권 / 예금·적금 | 이자 | 이자소득 ✅ |
| 리츠(REITs) | 배당 | 배당소득 ✅ |
| 분리과세·비과세(ISA·연금저축·IRP) | 수익 | ❌ 금융소득 제외 (현금흐름에만) |

이 표가 도구의 핵심 가치다: **같은 6억이라도 배당주에 담으면 금융소득이 커져 종합과세·건보료가
오르고, 국내 주식형 ETF·저분배 상품에 담으면 금융소득이 작아 위험이 낮아진다.**

## 3) 계산 흐름

```
포트폴리오(상품 라인들)
  → portfolio_to_financial(): 유형별로 FinancialInput 조립 + 비과세 현금흐름 + 평가총액
  → 기존 엔진 재사용: calc_financial_income → 종합과세 판정 → calc_income_tax → calc_health_insurance
  → 목표 나이~90세 연 단위 반복 (연금 개시·사적연금 세율 변화·피부양자 전환 자동 반영)
  → 포트폴리오마다 결과를 모아 비교
```

## 4) 가정과 한계 (이 모드 고유)

- **양도차익(매도 차익)은 0으로 가정.** 분배금·배당·이자만 반영한다. 양도차익은 종합과세·건보료에
  거의 영향이 없고(해외 양도는 분리과세, 건보 미반영), 양도세만 별도 발생하므로 "종합과세·건보 리스크"
  분석이라는 목적에는 영향이 적다. (차후 확장: 연 실현 양도차익 입력칸 추가)
- **자산 성장 미가정.** 사용자가 목표 나이 평가액을 직접 입력한다. 매년 잉여 현금은 적립, 부족분은
  원금에서 인출하는 방식으로만 자산 잔액이 변한다(수익률에 의한 원금 성장 없음).
- **근로·사업소득 없음.** 노후 자산 중심 모드라 근로·사업소득은 받지 않는다. 임대(월세)와 연금은 반영.
- 나머지 세법·건보 단순화는 `docs/02` 9장(S1~S12)과 동일.

## 5) API

| | |
|--|--|
| `GET /api/product-types` | 상품 유형 드롭다운용 {코드: {label, note}} |
| `POST /api/simulate-portfolio` | 입력: basic(목표나이 등)·pension·property·portfolios[1~3] / 출력: 구성별 비교 결과 |

## 6) 차후 확장 후보

- 연 실현 양도차익 입력 → 양도세까지 포함한 총 세부담
- 자산 성장률(원금 상승) 토글
- 포트폴리오별 "금융소득 2,000만/1,000만 자동 맞춤" 제안 (목표 분배율 역산)
- ISA·연금계좌 납입 한도·인출 규칙 반영
